Prompt-based Learning in NLP

Posted on 2021-10-25 Edited on 2021-10-26

This blog is based on: Liu et al. (2021) Pre-train, Prompt, and Predict: A Systematic Survey of Prompting Methods in Natural Language Processing.

1 Introduction

In a traditional supervised learning system (e.g. text classification or machine translation), we take an input x, and predict an output y based ona model \(P(\textbf{y}|\textbf{x};\theta)\). The main issue with supervised learning tasks is the requirement of larger amount of annotated label, which are usually not suffcicient for many task. This can be addressed by prompt-based learning. Instead of learning \(P(\textbf{y}|\textbf{x};\theta)\), prompt-based learning attempts to learn the probablity \(P(\textbf{x};\theta)\) and use this probability to predict y. > After appropriate prompt engineering, label y is usually included in the prompt text. ## 2 Prompting Basics Basic prompting works in following three steps. ### 2.1 Prompt Addition In this step, a prompting function \(f_{\textit{prompt}}\) is applied to modify the input text x into a prompt \(x^{\prime}=f_{\textit{prompt}}(\textbf{x})\). Traditionally, the function consists of two steps: 1. Apply a template, which is a textual string that takes two slots: an input slot \([X]\) for input x and an answer slot \([Z]\) for an intermediate generated answer text z that will later be mapped into target y. 2. Fill slot \([X]\) with the input text x.

For example, in the case of sentiment analysis with x ='I love this movie', the template can be :"[X] Overall, it was a [Z] movie". Then, \(x^{\prime}\) will turn into "I love this movie. Overall it was a [Z] moview". In the context of machine translation. it may take the form as "Enligh: [X] French [Z]". > Cloze prompt: the answer slot is in the middle of the text. Prefix prompt: the input text comes entirely before z.

2.2 Answer Search

We search over the set of potential answers z by calculating the probability of their corresponding filled prompts using a pre-trainied LM \(P(\dot;\theta)\): \[\hat{\boldsymbol{z}}=\operatorname{search}_{\boldsymbol{z} \in \mathcal{Z}} P\left(f_{\mathrm{fill}}\left(\boldsymbol{x}^{\prime}, \boldsymbol{z}\right) ; \theta\right) \] where \(\mathcal{Z}\) is a set of all permissible values for slot [Z], \(f_{\mathrm{fill}}\) is a function that fills in the location [Z] in prompt \(x^{\prime}\) with the potential answer z. The search function could be an argmax or \(sampling\). ### 2.3 Answer Mapping The next step is obtain the highest score output \(\hat{\textbf{y}}\) from the highest score \(\hat{\textbf{z}}\). In language generation tasks, the answer itself is the output. But in sentimen analysis tasks, multiple answers could result in the same output.
